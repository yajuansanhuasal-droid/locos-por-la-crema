# Locos por la Crema 🍦

Bienvenido al repositorio oficial de **Locos por la Crema**. Aquí gestionamos el código de nuestra plataforma web.

## 🚀 Tecnologías
* HTML5 / CSS3
* JavaScript
* GitHub Pages (para el despliegue)

## 📂 Estructura del Proyecto
* `src/`: Contiene el código fuente.
* `assets/`: Imágenes y recursos visuales.

## 🛠️ Instalación
1. Clona el repo: `git clone https://github.com/tu-usuario/locos-por-la-crema.git`
2. Abre `index.html` en tu navegador.
